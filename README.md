# Império Fit - Landing Page de Academia

Landing page fictícia criada para uma academia, com foco em apresentação de planos, modalidades e contato via WhatsApp.

## Link do projeto

 https://pedropagl.github.io/site-academia-imperio-fit/

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Site responsivo
- Menu de navegação
- Seção de planos
- Botões de WhatsApp
- Seção de modalidades
- Formulário simples
- Design moderno para academia

## Objetivo

Esse projeto foi criado para treinar desenvolvimento web e servir como exemplo de site para pequenos negócios.

## Status

Finalizado como primeira versão.
